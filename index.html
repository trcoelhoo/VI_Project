Introdução

Todos os dados estatísticos apresentados foram retirados do site PORDATA, referentes aos anos de 2011 a 2021 para dados gerais de Portugal e referentes aos anos de 2017 a 2021 para dados de distritos.

Neste site podem ser explorados diferentes tipos de visualização dos dados, agrupados por região/distrito e por dados gerais de Portugal. Dados estes como receitas, recintos, sessões, espetadores, lotação e ecrâs.

Para mais informações sobre os dados pode aceder ao site da PORDATA.
